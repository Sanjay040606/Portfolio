# Sanjay Portfolio

This is a portfolio website for Sanjay S built with `Next.js`, `Tailwind CSS`, and `GSAP`.

## Stack

- `Next.js` for the application structure and routing
- `React` for UI composition
- `Tailwind CSS` for styling
- `GSAP` for intro and section animations

## Project Structure

- `app/layout.tsx` sets the root HTML layout and metadata
- `app/page.tsx` contains the full portfolio page content and interactions
- `app/globals.css` contains the global Tailwind import and base styles
- `public/sanjay-img.jpeg` stores the profile image
- `public/Sanjay-S-Resume.pdf` stores the downloadable resume

## Features

- Minimal portfolio layout inspired by the requested reference
- Hero section with profile image and social links
- About / story section
- Projects section
- Highlights section
- History timeline section
- Contact form that opens the user's mail app
- Resume download link
- GSAP-based entrance animation

## Run Locally

1. Open a terminal in `d:\Learning_area\Portfolio`
2. Install dependencies if needed:

```bash
npm.cmd install
```

3. Start development server:

```bash
npm.cmd run dev
```

4. Open `http://localhost:3000`

## Production Build

```bash
npm.cmd run build
npm.cmd run start
```

## Notes

- The contact form currently uses `mailto:` instead of a backend form service.
- If you want, this can be upgraded later with `Resend`, `EmailJS`, or `Formspree`.
- The current design is a close custom rebuild inspired by the reference portfolio structure, not a direct copy of its original assets or proprietary effects.
